# MockMate AI

MockMate AI is an AI-powered interview preparation platform built using the MERN stack and Gemini AI.

It helps users generate interview reports, analyze skill gaps, prepare personalized interview plans, and create ATS-friendly resumes automatically from uploaded resume PDFs.

---

# Features

- JWT Authentication
- Resume PDF Upload
- AI-generated Interview Reports
- Technical Interview Questions
- Behavioral Interview Questions
- Skill Gap Analysis
- Personalized Preparation Plan
- AI-generated ATS-Friendly Resume PDF
- MongoDB Database Integration
- Secure Protected Routes
- PDF Parsing and Processing
